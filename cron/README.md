# Paradox of Acceptance — Weekly Digest Cron

Vercel cron job that sends a weekly community digest email every Sunday at 9am CT.

## What it sends

- Top 3 community posts from the past 7 days (ranked by reply count)
- Top 3 Q&A questions from the past 7 days
- Branded with the Paradox of Acceptance design system

## Cron schedule

`0 14 * * 0` — Sunday 14:00 UTC = 9:00am CT

Configured in `vercel.json`.

## Setup

### 1. Supabase schema

Run `schema.sql` in Supabase SQL Editor. Creates:

- `community_posts` — discussion threads with reply counts
- `community_qa` — Q&A pairs
- `email_sends` — automated send log

### 2. Deploy to Vercel

```bash
cd cron
npm install
vercel deploy --prod
```

Point Vercel at this directory as the project root.

### 3. Environment variables (set in Vercel dashboard)

| Variable                  | Required | Description                              |
|---------------------------|----------|------------------------------------------|
| `CRON_SECRET`             | Yes      | Auto-generated by Vercel; also set here  |
| `RESEND_API_KEY`          | Yes      | Resend API key                           |
| `RESEND_AUDIENCE_ID`      | Yes      | Resend audience (subscriber list)        |
| `EMAIL_FROM`              | Yes      | Verified sender address                  |
| `EMAIL_REPLY_TO`          | No       | Reply-to header                          |
| `SUPABASE_URL`            | No       | Enables community data + logging         |
| `SUPABASE_SERVICE_ROLE_KEY` | No     | Required if SUPABASE_URL is set          |

## Graceful skip conditions

The cron returns `200 { skipped: true }` without sending if:

- `RESEND_API_KEY` is not set
- `RESEND_AUDIENCE_ID` is not set
- Fewer than 2 community posts this week (nothing interesting enough to send)

## Local testing

```bash
cp .env.example .env   # fill in real values
npm install
vercel dev             # starts local dev server at localhost:3000

# Trigger the cron manually:
curl -H "Authorization: Bearer $CRON_SECRET" http://localhost:3000/api/cron/weekly-digest
```
